# DCS Web-Tac

A B/S architecture flight data analysis and playback tool for DCS World.

## Features
- Flight data ingestion via `Export.lua`.
- Web-based 3D playback (planned).
- Detailed flight performance metrics.

## Tech Stack
- **Backend**: Python (FastAPI)
- **Database**: SQLite
- **Frontend**: Vue.js / Cesium.js (planned)
